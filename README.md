# Простейший калькулятор 

#### Александр Голубин, Хайдаров Данил, Комарков Константин
--- 

Необходимо реализовать простейший калькулятор, который будет принимать 2 числа и какую либо операцию (умножение, вычитание, сложение, деление), после чего в соответствии с выбранной операцией совершает вычисления результат которых выводит пользователю. Также необходимо учесть возможные ошибки по типу деления на ноль или ввода пользователем символов вместо чисел. Каждая операция должна быть в отдельном методе.

Тест кейсы:

1) Ввод первой переменной(первого числа) "3"
Ввод второй переменной(второго числа) "2"
Выбор операции(+, -, *, /) "+"
Вывод результата "5"

2) Ввод первой переменной(первого числа) "3"
Ввод второй переменной(второго числа) "2"
Выбор операции(+, -, *, /) "*"
Вывод результата "6"

3) Ввод первой переменной(первого числа) "3"
Ввод второй переменной(второго числа) "2"
Выбор операции(+, -, *, /) "/"
Вывод результата "1,5"

4) Ввод первой переменной(первого числа) "3"
Ввод второй переменной(второго числа) "2"
Выбор операции(+, -, *, /) "-"
Вывод результата "1"

5) Ввод первой переменной(первого числа) "2"
Ввод второй переменной(второго числа) "3"
Выбор операции(+, -, *, /) "-"
Вывод результата "-1"

6) Ввод первой переменной(первого числа) "3"
Ввод второй переменной(второго числа) "-2.5"
Выбор операции(+, -, *, /) "*"
Вывод результата "-7.5"

7) Ввод первой переменной(первого числа) "30"
Ввод второй переменной(второго числа) "-30"
Выбор операции(+, -, *, /) "-"
Вывод результата "0"

8) Ввод первой переменной(первого числа) "3"
Ввод второй переменной(второго числа) "1.5"
Выбор операции(+, -, *, /) "-"
Вывод результата "2"

9) Ввод первой переменной(первого числа) "3"
Ввод второй переменной(второго числа) "0"
Выбор операции(+, -, *, /) "/"
Вывод результата "На ноль делить нельзя!"

10) Ввод первой переменной(первого числа) "а"
Ввод второй переменной(второго числа) "2"
Выбор операции(+, -, *, /) "+"
Вывод результата "Ошибка! Неверный тип данных"
